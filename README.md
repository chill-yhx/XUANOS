## 当前目标

完成视觉规范、页面线框和静态前端原型。

## 文档目录说明

- `docs/00_product/`：产品范围、MVP 边界与阶段目标。
- `docs/01_design/`：视觉风格指南、页面线框与界面设计规范。
- `docs/02_system_protocols/`：系统协议、会话流程、计划决策协议等核心规格文档。
- `docs/archive/`：历史资料与原始规格包备份。
- `design/`：后续设计源文件、素材与导出资源。
- `research/`：用户研究、竞品分析、技术调研与实验记录。
- `frontend/`：静态前端原型与后续前端工程代码。
- `backend/`：后续后端服务、接口与数据层代码。

## 下一步开发顺序

1. 固化 MVP 阶段 1 的产品范围与核心页面清单。
2. 根据视觉规范和页面线框搭建静态前端原型。
3. 补齐关键交互状态、响应式布局和基础组件。
4. 梳理前后端接口草案与数据结构。
5. 进入 MVP 阶段 2 的功能联调与可用性验证。
